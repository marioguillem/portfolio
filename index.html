  2. Analista de datos - Aitana Management

    Enero 2024 - Actualidad

    Mi trabajo como Analista de datos para la consultora tecnológica Aitana Management se basa en ofrecer soluciones de análisis de datos a los clientes mediante las herramientas de Power BI. Para clientes que requieren soluciones de ingeniería de datos aporto soluciones mediante Microsoft Fabric, desde procesos automatizados ETL, hasta la creación de Data Warehouses, Data Lakes o Lakehouses. Para la gestión de versiones utilizo Git y Azure DevOps.
